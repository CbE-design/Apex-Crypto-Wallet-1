# **App Name**: Apex Crypto Wallet

## Core Features:

- Portfolio Overview: Display the user's current cryptocurrency holdings and their values.
- Simple Buy/Sell Interface: Allow users to quickly buy and sell cryptocurrencies using a simplified interface.
- Market Overview: Display a list of top cryptocurrencies and their current prices.
- Crypto News Summary: Provide a short, AI-generated summary of the latest news related to the user's portfolio and the top cryptocurrencies. The AI tool will extract relevant news, remove hype or shilling, and produce neutral summary.
- Transaction History: Display the history of the users cryptocurrency transactions.
- Price Alerts: Alert users when the price of one of their cryptocurrencies reaches a threshold

## Style Guidelines:

- Primary color: Vibrant blue (#4285F4) for a trustworthy and modern feel.
- Background color: Light grey (#F5F5F5) for a clean, uncluttered backdrop.
- Accent color: Electric Blue (#7DF9FF) to highlight calls to action, create interest, and further enhance a tech feel.
- Font: 'Inter', a grotesque-style sans-serif for clear and neutral UI elements.
- Use modern, minimalist icons representing different cryptocurrencies and functions.
- Clean and intuitive layout with clear separation of sections for easy navigation.
- Subtle transitions and animations to enhance user experience during interactions.
